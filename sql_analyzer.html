Adatbázis Modell Dokumentáció & További SQL Példák

Táblák és Oszlopok

Az in-memory adatbázis 3 fő táblából áll:

  • fokategoriak: A legfelső szintű kódok aggregált adatai (pl. 90-00-00 AUTOMATIKA).
  • alkategoriak: A középső szintű mappák aggregált adatai (pl. 90-40-00 Kábelek).
  • tetelek: A tényleges, árazott fizikai tételek. Ez az alapértelmezett tábla.

Elérhető oszlopok a 'tetelek' táblában:

Oldal: PDF oldalszám
Kod: Tétel azonosító
Leiras: Megnevezés és megjegyzés
Mennyiseg: Mennyiség
Egyseg: Mértékegység
Anyag_ear: Anyag egységár
Dij_ear: Díj egységár
Anyag_ossz: Anyag összesen
Dij_ossz: Díj összesen
Total_ossz: Anyag + Díj összesen
Fokategoria_Nev: A tétel szülő kategóriája

Hasznos Haladó Lekérdezések

1. Átlagos négyzetméterárak főkategóriánként

Megmutatja, melyik részterületen dolgoztak a legmagasabb fajlagos m2 áron.

SELECT Fokategoria_Nev, ROUND(AVG(Total_ossz / Mennyiseg), 0) AS Atlag_m2_ar FROM tetelek WHERE Egyseg = 'm2' AND Mennyiseg > 0 GROUP BY Fokategoria_Nev ORDER BY Atlag_m2_ar DESC

2. Legdrágább egységárú tételek (Anyag)

Pusztán a legdrágább fizikai elemek beazonosítása (pl. egyedi üvegek, gépek).

SELECT Kod, Leiras, Anyag_ear, Egyseg FROM tetelek ORDER BY Anyag_ear DESC LIMIT 15

3. Mennyiségek eloszlása alkategóriákban

Megmutatja, miből használták a legtöbb folyómétert (pl. csövek, kábelek).

SELECT Alkategoria_Nev, SUM(Mennyiseg) AS Ossz_Mennyiseg, MAX(Egyseg) AS Mertek FROM tetelek WHERE Egyseg = 'm' GROUP BY Alkategoria_Nev ORDER BY Ossz_Mennyiseg DESC LIMIT 15

4. Legdrágább teljes árú (Anyag + Díj) tételek

A legnagyobb volumenű egyedi tételek a teljes projektben.

SELECT Kod, Leiras, Total_ossz, Anyag_ossz, Dij_ossz FROM tetelek ORDER BY Total_ossz DESC LIMIT 15

5. Anyagköltség nélküli, magas munkadíjú tételek

Olyan munkák, ahol csak munkadíj került felszámításra (1 millió Ft felett).

SELECT Kod, Leiras, Dij_ossz, Fokategoria_Nev FROM tetelek WHERE (Anyag_ossz = 0 OR Anyag_ossz IS NULL) AND Dij_ossz > 1000000 ORDER BY Dij_ossz DESC LIMIT 15
